# MyShop grocery app

**MyShop** is a grocery application that offers users product browsing, and shopping cart management, all within a highly responsive and interactive user interface.

### [Live Demo](https://my-shop-web.pages.dev/)

## Technologies Used

- React.js
- React Router(v6)
- JS
- Fetch, REST API
- CSS
- useSWR
